Common Commands

Create Command
(DDL-Data Definition Language)
CREATE TABLE table_name (column_name1 datatype(size), column_name2 datatype(size), column_name3 datatype(size));

Drop Command
DROP TABLE table_name;

Alter Command

Syntax to adda column into a table:
ALTER TABLE table_name ADD (column_name datatype(size));

Syntax to add a primary key to a table:
ALTER TABLE table_name ADD primary key (column_name);

Truncate Command (Remove all records from a table)
TRUNCATE TABLE table_name;

Syntax to COMMENT a line in SQL:

--Retrieve all data from a table SELECT * FROM table_name;
